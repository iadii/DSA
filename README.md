## 📂 Project Structure

### 🧱 1. [Basic C++](file:///Users/adiivats/Desktop/Adii/codes/DSA/01_basic%20cpp)
Fundamental concepts for mastering C++.
- **Basic Concepts**: Variables, operations, and memory management.
- **Patterns & Loops**: Building logic through pattern printing and iterative structures.
- **Core Problems**: Factorial, number reversal, and more.

### 📐 2. [Arrays & Matrices](file:///Users/adiivats/Desktop/Adii/codes/DSA/02_Array)
Deep dive into linear and multi-dimensional data structures.
- **Vector Implementation**: Dynamic arrays, resizing, and optimizations.
- **Matrix Operations**: Transpose, search, and diagonal manipulations.
- **LeetCode Gems**: Curated problem set from LeetCode.

### 🍱 3. [STL (Standard Template Library)](file:///Users/adiivats/Desktop/Adii/codes/DSA/03_STL)
Leveraging the power of C++ containers.
- **Pairs & Tuples**: Grouping related data.
- **Vectors & Lists**: Mastering dynamic sequences.

### 🔍 4. [Searching & Sorting](file:///Users/adiivats/Desktop/Adii/codes/DSA/04_SearchSort)
The backbone of efficient algorithms.
- **Searching**: Linear Search, Binary Search (and its variants).
- **Sorting**: Bubble Sort, with more algorithms being added.

### 🔢 5. [Bitwise Operations](file:///Users/adiivats/Desktop/Adii/codes/DSA/05_Btwise)
Low-level optimizations and bit manipulation tricks.
- **Set/Reset Bits**, **XOR Riddles**, **Even/Odd Detection**.

### 📝 6. [Practice Arena](file:///Users/adiivats/Desktop/Adii/codes/DSA/06_practice)
Dedicated space for reinforcing concepts through rigorous practice.

## ⚙️ Getting Started

### Prerequisites
Ensure you have a modern C++ compiler installed:
- **GCC/G++** (version 7.0 or higher)
- **Clang** (version 5.0 or higher)

### 🛠️ How to Run

1. **Clone the repository**:
   ```bash
   git clone https://github.com/adiivats/DSA.git
   ```
2. **Navigate to the directory**:
   ```bash
   cd DSA/02_Array
   ```
3. **Compile the code**:
   ```bash
   g++ linearSearch.cpp -o linearSearch
   ```
4. **Execute**:
   ```bash
   ./linearSearch
   ```
